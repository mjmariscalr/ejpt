# Preparación para el Examen eJPT

Este repositorio contiene la información y recursos que he usado para la preparación del examen eJPT (eLearnSecurity Junior Penetration Tester). Este es un proyecto personal que documenta mi progreso y aprendizaje mientras me preparo para obtener esta certificación. Aquí se encuentran una serie de contenidos teóricos y prácticos recopilados de distintas fuentes que he usado para superar el examen.

## Estructura del Curso

El curso está organizado en varios **módulos**, cada uno enfocado en un área específica de la ciberseguridad y el hacking ético. Dentro de cada módulo, encontrarás:

- **Teoría**: Explicaciones de los conceptos fundamentales.
- **Ejercicios prácticos**: Actividades y laboratorios para aplicar los conocimientos adquiridos.

## Resultados Esperados

Al finalizar este curso, espero haber adquirido las habilidades necesarias para:

- Realizar un pentest básico en una red o aplicación web.
- Utilizar herramientas de hacking ético y realizar análisis de vulnerabilidades.
- Obtener la certificación eJPT al aprobar el examen oficial.
