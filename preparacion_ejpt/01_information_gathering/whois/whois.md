# WhoIs

WHOIS es un protocolo para efectuar consultas en una base de datos que permite determinar el propietario de un nombre de dominio o una dirección IP en Internet. Las consultas WHOIS se pueden realizar bien a través de una utilidad para línea de comandos, o bien a través de una multitud de páginas web públicas que permiten realizar estas consultas.

## Guía de uso

Para usar esta herramienta basta con acceder a una de las múltiples webs e introducir una URL o una dirección IP. En linea de comandos lo hacemos de la misma forma.

```bash
$ whois wikipedia.com
```

## Riesgo de detección

El riesgo de detección en una consulta whois es prácticamente nulo, ya que solo pide información pública a una base de datos también pública.
