# Wappalyzer

Es una herramienta que muestra una lista con todas las tecnologías con las que está construida una web. Está disponible como extensión para varios navegadores con una visualización clara de la información. Wappalyzer analiza un dominio y revela detalles como:

- Lenguajes y frameworks.
- Servidores web.
- Sistemas de gestión de contenido.
- Servicios de terceros integrados.
- Tecnologías de seguridad.
- Plataformas e-commerce.
- Sistemas de publicidad y tracking.

## Guia de uso

Para poder usar esta herramienta simplemente accedemos a la web a analizar con la extensión ya instalada y hacemos click sobre la extensión en la barra de herramientas.

![imagen](../../img/wappalyzer.png)

## Implicaciones legales y riesgo de deteccion

Wappalyzer es una herramienta pasiva y analiza tecnologías visibles públicamente, generalmente código fuente HTML accesible públiamente, encabezados HTTP Y recursos externos usados por el sítio. Todo esto lo haces sin interactuar de forma intrusiva. Aunque su riesgo de detección es bajo, las conexiones de rastreo realizadas por Wappalyzer pueden aparecer en logs, lo que puede generar sospechas de que el objetivo está siendo analizado.
